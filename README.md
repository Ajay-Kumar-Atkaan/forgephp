# ForgePHP

> ⚠️ **ForgePHP is an educational project.** Its goal is to explore framework architecture by implementing core features from first principles.

ForgePHP is a lightweight PHP MVC framework built from scratch for learning, experimentation, and understanding how modern web frameworks work.

## Features

- Front Controller
- Routing
- Dynamic Route Parameters
- Request
- Response
- Reflection-based Dependency Injection
- Controller Dispatch
